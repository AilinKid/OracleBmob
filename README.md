# Oracle
一个简易的安卓在线考试系统，基于bmob云
2种用户登录，实现不同的功能，老师：查看试卷，添加试卷，个人信息 学生：查看试题，做题，历史做题，个人信息
在SystemTestSystem中更改你注册的bmob云key就可以运行，或者直接用默认的，但是您不能查看到数据表</br>
